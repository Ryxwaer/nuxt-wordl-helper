**Headline (this supersedes last week's "trend is healthy" read):** there is a **ranking demotion that began on ~2026-05-18**. Daily impressions fell off a cliff from a 120–750/day baseline to ~10/day and have stayed there for 11+ days. Last week's report missed it because its 7-day window (May 16–22) straddled the cliff and its 28-day window was propped up by the strong early-May weeks.

This is **a ranking loss, not a demand collapse and not the game ending**: the Binance WODL game is live (the theme API returns `Binance AI Safety` with a full word pool), Cloudflare real-edge traffic is flat, and our average position only worsened modestly — but we **fell out of the head terms** (details in §3). The most probable cause is a post-deploy freshness "honeymoon" decay following the early-May deploys (a textbook spike-then-settle), with a mid-May Google update as the alternative. The May-24 code changes did **not** cause it (the slide predates them by ~6 days).

Separately, a strategic finding (§6): the SERP for this niche is won by **dated, daily-refreshed answer-content pages**, not tools — which is directly relevant to the AdSense goal and to our product principle.

## §6. Competitive landscape — who actually wins these SERPs

Pulled the live SERP for the core queries. The winners are **not tools** — they are dated, daily-updated answer-content pages:

| Competitor | What it is | Has a solver? |
|---|---|---|
| coingabbar.com | "Binance Word of the Day Answers Today **30 May 2026**: Full WOTD List", `Last Updated: 30-05-2026` | No |
| coinfantasy.io | Huge per-length answer list with today's answers flagged; deep how-to/points/tips | No — **links out** to iToolVerse + Miguel Fernandes |
| mexc.com, coinbrain.com, finestel.com, noluckgaming.com, earnenews.com | Dated daily/weekly answer articles | No |

**Key insight:** Google rewards *freshness + "today's answers" framing* for this niche. The standalone solver tools mostly get discovered **embedded inside** these content pages. We hold the two best raw cards — a genuinely live current-week pool **and a working native solver** (which coinfantasy, the strongest competitor, lacks) — but we were packaged like a static tool, not a fresh answer page.

**Tension with the product principle / AdSense goal:** the calculator principle ("tool, not content directory") is clean, but (a) it's how the SERP-winning pattern is *lost*, and (b) AdSense's most common rejection is "low-value / thin content" — a 3-page prose-light tool is exactly that profile. For the AdSense goal specifically, leaning into honest, **current-week, dated** answer content (never stale/archived pools — §6 of last week still stands on that) is the path that serves both ranking and approval. Decision pending; this report only flags it.
